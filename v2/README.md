# Installation

```bash
brew install rustup-init
```

#### After package manager install

```bash
rustup toolchain install stable
```

# Basic Data Types

- Boolean
    - true, false
- Integer
    - 1, 2, 50, 99, -2
- Double / Float
    - 1.1, 5.5, 200.0001, 2.0
- Character
    - 'A', 'B', 'c', '6', '$'
- String
    - "Hello", "string", "this is a string", "it's 42"
