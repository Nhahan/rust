# Rust

## Installation
```bash
# https://rustup.rs/
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# After run curl
source "$HOME/.cargo/env"
```

## Uninstall
```bash
rustup self uninstall
```

## Single File Compilation
```bash
rustc filename.rs
```

## Cargo (Package Manager & Build System)
```bash
cargo build
cargo run
```
